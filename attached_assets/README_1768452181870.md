# BioStar X Mapper - Suprema Inc.

![BioStar X Mapper](https://www.supremainc.com/es/images/logo.png)

**BioStar X Mapper** es una herramienta web de preventa técnica diseñada para simplificar y precisar el cálculo de licenciamiento de la plataforma BioStar X. Permite a integradores, distribuidores y personal de preventa dimensionar proyectos de forma rápida, obteniendo un Bill of Materials (BOM) exacto basado en las necesidades específicas de puertas, usuarios y funciones avanzadas.

## 🚀 Características Principales

- **Dimensionamiento en Tiempo Real**: Calcula el tier de licencia adecuado (Starter, Essential, Advanced, Enterprise o Elite) mientras se ingresan los datos.
- **Doble Escenario**:
  - **Proyecto Nuevo**: Para implementaciones desde cero.
  - **Migración BioStar 2**: Flujo especializado para actualizaciones de sistemas legacy, incluyendo validación de hardware y evidencias.
- **Categorización Avanzada**:
  - **Advanced AC**: Selección modular de funciones como Global APB, Interlock, Mustering, etc.
  - **Módulos y Add-ons**: Visitantes, T&A, Mobile Access, Mapas Gráficos, Directorio Activo (AD/LDAP).
  - **Licencias de Dispositivo**: Gestión de canales de video, cámaras QR y cerraduras inalámbricas.
- **Reporte Maestro**: Genera un documento técnico detallado listo para imprimir o enviar PDF, con toda la información del proyecto y el BOM.
- **Diseño Premium**: Interfaz moderna, responsiva y orientada a la experiencia de usuario (UX).

## 🛠️ Tecnologías

Este proyecto está construido para ser 100% portable y autónomo:
- **Core**: React 18 & Babel (vía CDN).
- **Estilos**: Tailwind CSS.
- **Fuentes**: Google Fonts (Montserrat & Noto Sans).
- **Integridad**: Sin dependencias de instalación local; funciona directamente desde el navegador.

## 📂 Estructura del Proyecto

- `Borrador_Calculador_SupremaBioStarX_enero2026.html`: Archivo principal de la aplicación.
- `BioStarX_Mapper_Technical_Specification.md`: Documentación técnica detallada de la lógica de cálculo y diccionario de datos.
- `README.md`: Este archivo.

## 📖 Cómo Usar

1. Abre el archivo `Borrador_Calculador_SupremaBioStarX_enero2026.html` en cualquier navegador moderno (Chrome, Edge, Firefox, Safari).
2. Selecciona el escenario del proyecto (Nuevo o Migración).
3. Completa los datos de identificación y contacto.
4. Ingresa las capacidades (Usuarios, Puertas, etc.).
5. Activa las funciones requeridas en la sección de Add-ons.
6. Observa el BOM generado dinámicamente en la barra lateral derecha.
7. Haz clic en **GENERAR REPORTE MAESTRO** para obtener el resumen final.
